# URL Shortener Project

## 📌 Description
This is a Spring Boot based URL Shortener project that converts long URLs into short links.

## 🚀 Features
- Generate short URL
- Redirect to original URL
- Track click count

## 🛠️ Tech Stack
- Java
- Spring Boot
- MySQL
- JPA / Hibernate
- Redish

## ▶️ How to Run
1. Clone the repository
2. Open in VS Code or IntelliJ
3. Run the Spring Boot application
4. Test APIs using Postman
